## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/psynexa-portfolio.git
   cd psynexa-portfolio
    ```
   
3. Install dependencies:

  ```
  npm install
  ```
3. Run the client
  ```
  npm run dev
  ```
